<alloca.h>

[non-standard]

Include the header <alloca.h> to define the function alloca, to allocate stack memory.

alloca

void *alloca(size_t size);

The function allocates size bytes of space in the stack frame of the caller. This temporary space is automatically freed when the function that called alloca returns to its caller. The function returns a pointer to the beginning of the allocated space. If the allocation causes stack overflow, program behavior is undefined.
